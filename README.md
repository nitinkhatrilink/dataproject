# Data Warehouse & Analytics Project

## Overview
This project focuses on building a simple data warehouse and performing analytical queries on structured data.

The goal is to practice core data analytics and data engineering concepts such as:
- Data modeling
- ETL workflows
- SQL analytics
- Business insights

---

## Project Components

- **Data Warehouse**  
  A structured schema designed to store analytical data.

- **ETL Process**  
  Scripts used to extract, transform, and load data into the warehouse.

- **SQL Analysis**  
  Queries used to explore and analyze the data.

---

## Tech Stack

- SQL (PostgreSQL)
- Python
- Power BI
